Analytics

Qualitative analysis is not only about creating codes and finding them in the texts. We offer advanced analytical tools to perform topic modelling, lexical, sentiment and network analysis.

Machine learning

Machine learning brings a quantitative touch to your qualitative research. It offers effective tools to find patterns and suggests appropriate codes for your key words.

Visualisations

Visualisaitons is a powerful tool that allow to see and understand data patterns. Codercats offers word clouds, graphs and diagrams to support your analytics and present results.

Collaboration

Codercats is a web-based application andyou can use it anywhere. It is the firstanalytical application that enables collaboration that is convenient for research groups to participate and monitor the progress.

Data formats and API

Codercats support a wide rangeof data formats. They include .doc, .pdf, .txt, .xls, .csv, and .html. You can also source your data using API tools.
